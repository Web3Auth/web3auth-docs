## Plug and Play SDKs Whitelabeling

Web3Auth's Plug and Play SDKs and WalletServicesPlugin offer whitelabeling capabilities via the
Dashboard (version 8.5.0 onwards)and code. The main whitelabeling options include

- Customize the Login Modal
- User Flow and MFA Screen Transition
- Language Customization and Localization

![Web3Auth Whitelabeling Dashboard](/images/whitelabel/whitelabel-dashboard.gif)

:::tip

To configure whitelabeling through code, refer to the
[Web3Auth PnP SDK Reference](/sdk/pnp/web/modal/whitelabel) for detailed guidance on customizing
your Plug-and-Play integration.

:::

:::note

Access to Whitelabeling is gated. The minimum [pricing plan](https://web3auth.io/pricing.html) to
use this feature is the **Growth Plan**.

:::

### Login Modal Customization

Adapt the login modal's design to reflect your application's branding, from colors and logos to
fonts and button styles.

### Language Customization and Localization

Web3Auth supports multiple languages, improving accessibility and user engagement across diverse
demographics.

- **Multi-Language Support:** Web3Auth offers localization in various languages, including English
  (en), Spanish (es), German (de), Chinese (zh), Japanese (ja), Korean (ko), French (fr), Portuguese
  (pt), and Dutch (nl).

  ![Language Modified Screen](@site/static/guides/whitelabel-web3auth-openlogin-language.png)

- **Open-Source Contributions:** Language support is maintained in an open-source repository,
  enabling community contributions to add or enhance translations.

  :::tip

  [Contribute to Web3Auth Locales](https://github.com/Web3Auth/web3auth-locales).

  :::
