# Twoslash TypeScript Integration Demo

This page demonstrates the Twoslash integration for enhanced TypeScript code blocks with interactive
type information.

## Basic Type Inference

```typescript twoslash
const userName = "Alice";
//    ^?

//highlight-start
const userAge = 30;
//    ^?
//highlight-end
```

## Simple Variable Test

```typescript twoslash
const message = "Hello World";
//    ^?

const number = 42;
//    ^?
```

## Function Return Type

```typescript twoslash
function add(a: number, b: number) {
  return a + b;
}

const result = add(5, 3);
//    ^?
```

## Usage Instructions

To use Twoslash in your documentation:

1. **Add `twoslash` to the code block meta**:

   ````markdown
   ```typescript twoslash
   const data = { message: "Hello!" };
   //    ^?
   ```
   ````

2. **Use `^?` comments** to show type information on hover
3. **Only blocks with `twoslash` will be processed** - regular code blocks remain unchanged
4. **Supports TypeScript, JavaScript, JSX, and TSX**

The integration provides:

- ✅ **Hover tooltips** with type information
- ✅ **Error highlighting** for TypeScript errors
- ✅ **Enhanced syntax highlighting**
- ✅ **Dark/light mode support**
- ✅ **No impact on regular code blocks**
