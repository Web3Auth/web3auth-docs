### Basic Details

#### Adapter Name: `wallet-connect-v1`

#### Package Name: `@web3auth/wallet-connect-v1-adapter`

#### authMode: `DAPP`

#### chainNamespace: `EIP155`

#### Default: `YES`

#### Description:

Wallet connect v1 adapter allows you login with wallet connect v1. You can read more about wallet connect [here](https://docs.walletconnect.com/).

### Interface

---

```ts
interface WalletConnectV1AdapterOptions {
  adapterSettings?: IWalletConnectProviderOptions;
  chainConfig?: CustomChainConfig;
}
```

### Params

---

| Parameter       | type                                                                                                         |
| --------------- | ------------------------------------------------------------------------------------------------------------ |
| chainConfig     | [CustomChainConfig](#common_configuration)                                                                   |
| adapterSettings | [IWalletConnectProviderOptions](https://docs.walletconnect.com/quick-start/dapps/client#initiate-connection) |

### Example

---

```ts
   import `Web3Auth` class from  `@web3auth/web3auth`;
   import { WalletConnectV1Adapter } from '@web3auth/wallet-connect-v1-adapter';
   const web3auth = new Web3Auth(options);
   const adapter = new WalletConnectV1Adapter({
       adapterSettings: {
        infuraId:"YOUR_INFURA_ID"
      },
      chainConfig:{
        chainNamespace: CHAIN_NAMESPACES.EIP155,
        chainId: "0x3",
        rpcTarget: "https://ropsten.infura.io/v3/776218ac4734478c90191dde8cae483c",
        displayName: "ropsten",
        blockExplorer: "https://ropsten.etherscan.io/",
        ticker: "ETH",
        tickerName: "Ethereum",
      }
  })

  web3auth.configureAdapter(adapter);
```
