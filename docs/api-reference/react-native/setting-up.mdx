## Installation

---

### Web3Auth React Native SDK

#### [`web3auth-react-native-sdk`](https://github.com/web3auth/web3auth-react-native-sdk)

```sh
npm install @web3auth/react-native-sdk
```

We will also require a `WebBrowser` implementation to allow our JS-based SDK to interact with the native APIs, and there are different extra
installation steps depending on whether you are using the bare workflow or managed workflow. Please refer to `Choosing between workflows` to pick the
correct implementation.

###

#### Android

- Perform the native [Android integration steps](/api-reference/android/setting-up).
- For Android, the `redirectUrl` parameter is configurable, and has to be added into the `AndroidManifest.xml`.

#### iOS

- Perform the native [iOS integration steps](/api-reference/ios/setting-up).

- Nothing has to be done with the iOS `Info.plist` since the `redirectUrl` is fixed to be `\(bundleId)://auth`, and does not need to be added as a iOS
  Custom URL Scheme.

##### Register the URL scheme you intended to use for redirection

- Android `AndoidManifest.xml` (required)
- iOS `Info.plist` (optional)

### Openlogin Expo SDK

Note that OpenLogin is the name of Web3Auth before the rebranding. We are currently working on rebranding our Expo offering.

#### Using a Managed Workflow

#### [`openlogin-expo-sdk`](https://github.com/torusresearch/openlogin-expo-sdk)

##### Install the `expo-web-browser` package

```sh
expo install expo-web-browser
```

##### Install the npm package

```sh
npm install openlogin-expo-sdk
```

To allow the Expo based SDK to work with exported Expo Android apps, you need to add the designated scheme into `app.json`

```json
{
  "expo": {
    "scheme": "openloginexposdkexampleexpo"
  }
}
```

:::tip

You may refer to [this example app](https://github.com/torusresearch/openlogin-expo-sdk/tree/master/openlogin-expo-sdk-example-expo) for more details.

:::

#### Using a Bare Workflow

#### [`openlogin-expo-bare-sample`](https://github.com/torusresearch/openlogin-expo-bare-sample)

Follow the instructions on the [Expo Documentation](https://docs.expo.dev/bare/installing-expo-modules/) to install the Expo SDK into your bare React
Native app.

##### Install the `expo-web-browser` package

```sh
expo install expo-web-browser
```

Follow the extra instructions in [Expo Documentation](https://github.com/expo/expo/tree/main/packages/expo-web-browser) for installing the
`expo-web-browser` package.

##### Install the Expo based SDK

```sh
npm install openlogin-expo-sdk
```

##### If you are using iOS, install the iOS dependecies

```sh
npx pod-install
```

:::tip

Register the URL scheme you intended to use for redirection into your Android `AndoidManifest.xml` (required) and iOS `Info.plist` (optional).

:::

## Initialisation

---

### Create an instance

#### Web3Auth React Native SDK using the `init()` function

```typescript
import Web3AuthReactNativeSdk, { LoginProvider, Web3AuthNetwork } from "web3auth-react-native-sdk";

await Web3AuthReactNativeSdk.init({
  // Your clientId obtained from Web3Auth dashboard.
  clientId: "BKJ3HmEqVmMHbFeW6E-CVPmdnVrnPhdBEI82kxgBVJGtaS4XlylvAE-1gmsv_Fa1CDj-xIhvTf3Kgd6mTn8nJtw",
  network: Web3AuthNetwork.MAINNET,
  // redirectUrl only applies for Android SDK, it is designated by iOS SDK in iOS, which is \(bundleId)://auth
  redirectUrl: "com.example.web3authreactnativesdk://auth",
});
```

#### Expo based SDK with bare or managed workflow

```typescript
import OpenLogin, { Network } from "openlogin-expo-sdk";
const openlogin = new OpenLogin({
  // Your clientId obtained from Web3Auth dashboard.
  clientId: "BC5bANkU4-fil7C5s1uKzRfF0VGqbuaxDQiLnQ8WgF7SEA32lGegAhu7dk4dZf3Rk397blIvfWytXwsRvs9dOaQ",
  network: Network.TESTNET,
});
```

#### `Web3AuthOptions` // `OpenLoginOptions` object

The `init()` method on in the Bare React Native SDK and the `OpenLogin` constructor in the Expo based SDK take an `OpenLoginOptions` or
`Web3AuthOptions` object respectively as an argument. The fields of such object are listed below.

| Parameter   | Type                               | Mandatory                    | Description                                                                                                                                                                                     |
| ----------- | ---------------------------------- | ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| clientId    | String                             | Yes                          | Your Web3Auth project ID obtained from the Web3Auth dashboard                                                                                                                                   |
| network     | enum of Web3AuthNetwork or Network | Yes                          | Web3Auth network to run on, either MAINNET or TESTNET                                                                                                                                           |
| redirectUrl | String                             | Yes on bare React Native SDK | The redirectUrl that the Web3Auth login flow will use to redirect back to the app. It is not required to be specified in the Expo based SDK since it will be specified in the `login()` method. |

## Example

---

```typescript
import Web3AuthReactNativeSdk, { LoginProvider, Web3AuthNetwork } from "web3auth-react-native-sdk";

React.useEffect(() => {
  Web3AuthReactNativeSdk.init({
    // Your clientId obtained from Web3Auth dashboard.
    clientId: "BKJ3HmEqVmMHbFeW6E-CVPmdnVrnPhdBEI82kxgBVJGtaS4XlylvAE-1gmsv_Fa1CDj-xIhvTf3Kgd6mTn8nJtw",
    // TESTNET is currently broken on iOS.
    network: Web3AuthNetwork.MAINNET,
    // redirectUrl only applies for Android SDK, it is designated by iOS SDK in iOS, which is \(bundleId)://auth
    redirectUrl: "com.example.web3authreactnativesdk://auth",
  })
    .then((result) => console.log(`success: ${result}`))
    .catch((err) => console.log(`error: ${err}`));
}, []);
```
