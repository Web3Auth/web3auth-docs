Web3Auth's Social Logins and Email Login runs using the OpenLogin Flow. The whole OpenLogin user experience can also be whitelabeled using
`OpenLoginAdapter` settings.

<div>
  <img style={{ minWidth: "250px", maxWidth: "300px" }} src={Image2} alt="open login modal" />
</div>

## Arguments

---

`new OpenloginAdapter({network, clientId, whiteLabel?})`

Below are the parameters you can pass on to the `whiteLabel` option to whitelabel your instance.

| Parameter for `whiteLabel` | Type      | Options                                                                                                                                                                                                      |
| -------------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `dark`                     | `boolean` | Can be set to `true` or `false` with default set to `false`.                                                                                                                                                 |
| `theme`                    | `object`  | Theme is a record of colors which can be configured. As of, now only `primary` color can be set and has effect on OpenLogin screens (default primary color is `#0364FF`). Theme affects the icons and links. |
| `defaultLanguage`          | `string`  | Default language will set the language used on all OpenLogin screens. The supported languages are: `en`, `de`, `ja`, `ko`, `zh`, `es`.                                                                       |

### `dark`

Can be set to `true` or `false` with default set to `false`.

## Example

---

```js
import { Web3Auth } from "@web3auth/web3auth";
import { OpenloginAdapter } from "@web3auth/openlogin-adapter";
import { CHAIN_NAMESPACES } from "@web3auth/base";

const web3auth = new Web3Auth({
  chainConfig: {
    chainNamespace: CHAIN_NAMESPACES.EIP155,
  },
  clientId: "example-client-id",
});

const openloginAdapter = new OpenloginAdapter({
  adapterSettings: {
    network: "testnet", // "testnet", "mainnet" or "cyan"
    clientId: "example-client-id",
    // type WhiteLabelData
    whiteLabel: {
      dark: true, // true or false
      theme: { primary: "#FFA500" },
      defaultLanguage: "en",
    },
  },
});

web3auth.configureAdapter(openloginAdapter);
```
