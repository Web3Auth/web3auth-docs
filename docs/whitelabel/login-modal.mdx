Web3Auth Plug and Play Modal is the UI that connects your dApp with all the Login Methods and Adapters that Web3Auth offers.

## Arguments

---

`new Web3Auth({clientId, chainConfig, uiConfig?})`

Below are the parameters you can pass on to the `uiConfig` option to whitelabel your instance.

<div style={{ flexBasis: "300px", flexGrow: "1", textAlign: "center" }}>
  <img src={Image1} style={{ alignSelf: "center", maxWidth: "90%" }} alt="UIConfig arguments" />
</div>

| Parameter for `uiConfig` | Type     | Options                                                                                                                                                                        |
| ------------------------ | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `theme?`                 | `string` | Can be set to `dark` or `light` with default to `light`.                                                                                                                       |
| `appLogo?`               | `string` | A link to your preferred logo and must be compatible with either `light` or `dark` mode. Our default logo is [web3auth-logo.svg](https://images.web3auth.io/web3auth-logo.svg) |
| `loginMethodsOrder?`     | `array`  | The list of login methods can be reordered with this parameter. Methods specified will be first on the list. e.g. `['facebook', 'google']`                                     |
| `defaultLanguage?`       | `string` | The language that will be used by web3auth Modal. [`en`: english, `de`: german, `ja`: japanese, `ko`: korean, `zh`: mandarin, `es`: spanish]                                   |

## Example

---

### Web

```js
import { Web3Auth } from "@web3auth/modal";
import { CHAIN_NAMESPACES } from "@web3auth/base";

const web3auth = new Web3Auth({
  // type uiConfig
  // highlight-start
  uiConfig: {
    appLogo: "https://images.web3auth.io/web3auth-logo-w.svg",
    theme: "dark",
    loginMethodsOrder: ["facebook", "google"],
    defaultLanguage: "en",
  },
  // highlight-end
  chainConfig: { chainNamespace: CHAIN_NAMESPACES.EIP155 },
  clientId: "example-client-id",
});
```

### Android

```kotlin
web3Auth = Web3Auth (
  Web3AuthOptions (
    context = this,
    clientId = getString (R.string.web3auth_project_id),
    network = Web3Auth.Network.MAINNET,
    redirectUrl = Uri.parse ("{YOUR_APP_PACKAGE_NAME}://auth"),
    // Optional whitelabel object
    // highlight-start
    whiteLabel = WhiteLabelData (
      name = "Web3Auth Android Sample App",
      logoLight = "https://images.web3auth.io/web3auth-logo-w.svg",
      logoDark = "https://images.web3auth.io/web3auth-logo-w.svg",
      defaultLanguage = "en",
      dark = true,
      theme = hashMapOf (
        "primary" to "#229954"
      )
    )
    // highlight-end
  )
)
```

### Flutter

```dart
Future<void> initPlatformState() async {
  final themeMap = HashMap<String, String>();
  // highlight-next-line
  themeMap['primary'] = "#229954";

  Uri redirectUrl;
  if (Platform.isAndroid) {
    redirectUrl = Uri.parse('{SCHEME}://{HOST}/auth');
    // w3a://com.example.w3aflutter/auth
  } else if (Platform.isIOS) {
    redirectUrl = Uri.parse('{bundleId}://openlogin');
    // com.example.w3aflutter://openlogin
  } else {
    throw UnKnownException('Unknown platform');
  }

  await Web3AuthFlutter.init(Web3AuthOptions(
    clientId: "WEB3AUTH_CLIENT_ID",
    network: Network.testnet,
    redirectUrl: redirectUrl,
    // highlight-start
    whiteLabel: WhiteLabelData(
      name: "Web3Auth Flutter App",
      logoLight: "https://images.web3auth.io/web3auth-logo-w.svg",
      logoDark: "https://images.web3auth.io/web3auth-logo-w.svg",
      defaultLanguage: "en",
      dark: true,
      theme: themeMap)
    ));
    // highlight-end
}
```
