# Slope Adapter

---

import TabItem from "@theme/TabItem";

## [`@web3auth/slope-adapter`](https://npmjs.com/package/@web3auth/slope-adapter)

Slope adapter allows you to connect with slope wallet. You can read more about slope wallet here.(https://docs.slope.app/).

## Basic Details

---

#### Adapter Name: `slope`

#### Package Name: [`@web3auth/slope-adapter`](https://npmjs.com/package/@web3auth/slope-adapter)

#### authMode: `DAPP`

#### chainNamespace: `SOLANA`

#### Default: `YES`

## Installation

---

```bash npm2yarn
npm install --save @web3auth/slope-adapter
```

### Custom Chain Config

#### `chainConfig`

:::caution

While you can pass your `chainConfig`, it is not required since you can directly pass it over to the `Web3Auth`/ `Web3AuthCore` configuration while
instantiating it.

Read more about it in their respective sections:

- [`web3auth/web3auth`](/sdk/web/web3auth/initialize#adding-a-custom-chain-configuration)
- [`web3auth/core`](/sdk/web/core/initialize#adding-a-custom-chain-configuration)

:::

:::warning

If you do pass `chainConfig` in an Adapter, it overwrites the `chainConfig` passed over to the `Web3Auth`/ `Web3AuthCore` constructor.

:::

## Example

---

```ts
import { SlopeAdapter } from "@web3auth/slope-adapter";
const slopeAdapter = new SlopeAdapter({
  clientId: "YOUR_WEB3AUTH_CLIENT_ID",
});
web3auth.configureAdapter(slopeAdapter);
```
