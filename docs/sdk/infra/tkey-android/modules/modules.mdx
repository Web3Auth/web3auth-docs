The tKey Android SDK can do more with the help of modules. These modules give you extra features for
handling your tKey shares. You can use them to generate seed phrases, create passwords, or transform
them into a format that can be easily recalled or shared.

### [Private Keys Module](/sdk/infra/tkey-android/modules/private-keys)

[`PrivateKeysModule`](/sdk/infra/tkey-android/modules/private-keys): This module allows for the
storage of additional private keys.

### [Security Question Module](/sdk/infra/tkey-android/modules/security-questions)

[`SecurityQuestionModule`](/sdk/infra/tkey-android/modules/security-questions): This module allows
users to add or remove security questions and passwords to share.

### [Seed Phrase Module](/sdk/infra/tkey-android/modules/seed-phrase)

[`SeedPhraseModule`](/sdk/infra/tkey-android/modules/seed-phrase): This module enables the storage
and use of seed phrases.

### [Share Serialization Module](/sdk/infra/tkey-android/modules/share-serialization)

[`ShareSerializationModule`](/sdk/infra/tkey-android/modules/share-serialization): This module
enables users to import/export shares from tKey.

### [Share Transfer Module](/sdk/infra/tkey-android/modules/share-transfer)

[`ShareTransferModule`](/sdk/infra/tkey-android/modules/share-transfer): This module allows for the
transfer of shares to other devices.
