The Web3Auth [tKey Android SDK](https://github.com/tkey/tkey-android) is a client-side library you
can use with your Android app to authenticate users using Web3Auth. This authentication can be
achieved using any social logins Web3Auth provides or a custom authentication flow. For using
Web3Auth's tKey natively in Android, Web3Auth provides a tKey Android SDK written in Java.

### Requirements

- Android API version `24` or newer.
- Basic knowledge of Java or Kotlin Development.

:::note

The minimum [pricing plan](https://web3auth.io/pricing.html) to use this SDK in a production
environment is the **Enterprise Plan**. However, you can use this SDK with all features enabled in
`sapphire_devnet` for free.

:::
