The Web3Auth's MPC [tKey MPC Swift SDK](https://github.com/tkey/tkey-mpc-swift) is a client-side
library you can use with your iOS app to authenticate users using Web3Auth. This authentication can
be achieved using any social logins Web3Auth provides or a custom authentication flow. For using
Web3Auth's tKey natively in iOS, Web3Auth provides a tKey iOS SDK written in Swift.

### Requirements

- iOS 13+
- Xcode 11.4+ / 12.x
- Swift 4.x / 5.x

:::note

The minimum [pricing plan](https://web3auth.io/pricing.html) to use this SDK in a production
environment is the **Enterprise Plan**. However, you can use this SDK with all features enabled in
`sapphire_devnet` for free.

:::
