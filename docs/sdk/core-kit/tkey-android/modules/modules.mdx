The tKey Android SDK can do more with the help of modules. These modules give you extra features for handling your tKey shares. You can use them to
generate seed phrases, create passwords, or transform them into a format that can be easily recalled or shared.

### [Private Keys Module](/sdk/core-kit/tkey-android/modules/private-keys)

[`PrivateKeysModule`](/sdk/core-kit/tkey-android/modules/private-keys): This module allows for the storage of additional private keys.

### [Security Question Module](/sdk/core-kit/tkey-android/modules/security-questions)

[`SecurityQuestionModule`](/sdk/core-kit/tkey-android/modules/security-questions): This module allows users to add or remove security questions and
passwords to share.

### [Seed Phrase Module](/sdk/core-kit/tkey-android/modules/seed-phrase)

[`SeedPhraseModule`](/sdk/core-kit/tkey-android/modules/seed-phrase): This module enables the storage and use of seed phrases.

### [Share Serialization Module](/sdk/core-kit/tkey-android/modules/share-serialization)

[`ShareSerializationModule`](/sdk/core-kit/tkey-android/modules/share-serialization): This module enables users to import/export shares from tKey.

### [Share Transfer Module](/sdk/core-kit/tkey-android/modules/share-transfer)

[`ShareTransferModule`](/sdk/core-kit/tkey-android/modules/share-transfer): This module allows for the transfer of shares to other devices.
