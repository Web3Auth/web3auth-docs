Via Web3Auth SDK, you can interact with [Ethereum](https://ethereum.org/) to get access to your `account`, fetch `balance`, `sign message`,
`send transaction`, and even `read` and `write` from the contract.

## Installation

---

To interact with the Ethereum blockchain, you can use either [web3.js](https://web3js.readthedocs.io/en/v1.2.8/getting-started.html) or
[ethers.js](https://docs.ethers.io/v5/getting-started/) libraries with Web3Auth.

<InstallationSnippet />

## Initializing Provider

---

Using `eip155` as `chainNamespace` while initializing `web3auth` will provide an `EIP1193` compatible provider as **`web3auth.provider`** after
successful authentication.

### Testnet

- Ropsten Block Explorer URL: https://ropsten.etherscan.io -- `chainId: "0x3"`
- Rinkeby Block Explorer URL: https://rinkeby.etherscan.io -- `chainId: "0x4"`
- Goerli Block Explorer URL: https://goerli.etherscan.io -- `chainId: "0x5"`

```js
/*
Simply change the chainId feild of `chainConfig` to `0x13881` (testnet)
*/

const web3auth = new Web3Auth({
  clientId: "YOUR_WEB3AUTH_CLIENT_ID", // get it from Web3Auth Dashboard
  chainConfig: {
    chainNamespace: "eip155",
    chainId: "0x3", // "0x4", "Ox5"
    rpcTarget: "https://rpc.ankr.com/eth_ropsten", // eth_rinkeby, eth_goerli
    displayName: "Ropsten Testnet",
    blockExplorer: "https://ropsten.etherscan.io", // rinkeby, goerli
    ticker: "ETH",
    tickerName: "Ethereum",
  },
});
```
