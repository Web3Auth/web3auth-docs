## Installation

Via Web3Auth SDK, you can interact with [Polygon](https://polygon.technology/). Polygon (formerly Matic) is EVM compatible so you can directly follow
the [Ethereum installation](./ethereum).

## Configure Polygon

### Mainnet

Mainnet Block Explorer URL: https://polygonscan.com

```js
import { Web3Auth } from "@web3auth/web3auth";
import Web3 from "web3";

const web3auth = new Web3Auth({
  clientId: "YOUR_CLIENT_ID", // get it from Web3Auth Dashboard
  chainConfig: {
    chainNamespace: "EIP155",
    chainId: "0x89", // hex of 137
  },
});
await web3auth.initModal();

const web3authProvider = web3auth.connect();

const web3 = new Web3(web3authProvider); // web3auth.provider
```
