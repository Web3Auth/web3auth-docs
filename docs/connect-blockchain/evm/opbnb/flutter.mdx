While using the Web3Auth Flutter SDK, you get the private key within the user scope. This private key can interact with
[opBNB](https://astar.network/) to make any blockchain calls, like getting the user's `account`, fetch `balance`, `sign transaction`,
`send transaction`, `read` from and `write` to the smart contract, etc. We have highlighted a few here to get you started quickly on that.
