After the user is login, you can retrieve the user's information by calling `torus.getUserInfo`:

```js
const userInfo = await torus.getUserInfo();
/*
  email: string;
  name: string;
  profileImage: string;
  verifier: string;
  verifierId: string;
 */
```

See [API reference](/wallet/api-reference/account#getuserinfo) for more details.
