---
title: Install Torus Embed SDK
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install Torus Embed SDK using a script tag, you can use either [jsdelivr](https://www.jsdelivr.com/package/npm/@toruslabs/torus-embed) or
[unpkg](https://unpkg.com/@toruslabs/torus-embed).

<Tabs
  defaultValue="jsdelivr"
  values={[
    { label: "jsdelivr", value: "jsdelivr" },
    { label: "unpkg", value: "unpkg" },
  ]}
>
<TabItem value="jsdelivr">

```html
<script src="https://cdn.jsdelivr.net/npm/@toruslabs/torus-embed"></script>
```

</TabItem>

<TabItem value="unpkg">

```html
<script src="https://unpkg.com/@toruslabs/torus-embed"></script>
```

</TabItem>

</Tabs>
