---
title: Whitelabel/customize brand logo and colors (optional)
---

You can also whitelabel or customize logo and colors to show your brand in the Wallet:

```js
await torus.init({
  // ...
  whitelabel: {
    theme: {
      isDark: false,
      colors: {
        torusBrand1: "#282c34",
      },
    },
    logoDark: "https://tkey.surge.sh/images/Device.svg", // Dark logo for light background
    logoLight: "https://tkey.surge.sh/images/Device.svg", // Light logo for dark background
    topupHide: false,
    featuredBillboardHide: true,
    disclaimerHide: true,
    defaultLanguage: "en",
  },
});
```

See [API reference](/wallet/api-reference/class#whitelabel) for more details.
