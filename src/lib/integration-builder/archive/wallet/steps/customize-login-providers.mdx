---
title: Customize login providers (optional)
---

You can customize login providers, e.g. using your Google/Facebook client ID instead of Torus's one.

```js
await torus.init({
  // ...
  loginConfig: {
    google: {
      clientId: "<YOUR GOOGLE CLIENT ID>",
      // ...
    },
  },
});
```

See [API reference](/wallet/api-reference/class#loginconfig) for more details.
