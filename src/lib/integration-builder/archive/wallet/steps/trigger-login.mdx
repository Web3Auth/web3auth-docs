---
title: Trigger user login
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Simply call `torus.login` to trigger user login wherever it makes sense in your application lifecycle:

<Tabs
  defaultValue="modal"
  values={[
    { label: "Modal selector", value: "modal" },
    { label: "Specific login", value: "specific" },
  ]}
>
<TabItem value="modal">

Calling `login` without any parameter will open a modal for user to select all supported logins:

```js
await torus.login();
```

</TabItem>

<TabItem value="specific">

You can also directly trigger a certain type of login by specifying a `verifier`:

```js
await torus.login({
  verifier: "reddit",
});
```

</TabItem>

</Tabs>

See [API reference](/wallet/api-reference/account#login) for more details.
