---
title: Instantiate the SDK and trigger login
---

Instantiate the SDK by creating an instance of `TorusSwiftDirectSDK` with login configurations of your choice then trigger user login when it's
appropriate in your application:

```swift
import TorusSwiftDirectSDK

let sub = SubVerifierDetails(loginType: .installed,
                            loginProvider: .google,
                            clientId: "<your-client-id>",
                            verifierName: <verifier-name>,
                            redirectURL: "<your-redirect-url>")
let sdk = TorusSwiftDirectSDK(aggregateVerifierType: <type-of-verifier>, aggregateVerifierName: <verifier-name>, subVerifierDetails: [sub])
sdk.triggerLogin(controller: <UIViewController>?).done{ data in
    print("Private key reconstructed:", data)
}.catch{ err in
    print(err)
}
```
