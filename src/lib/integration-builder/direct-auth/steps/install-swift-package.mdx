---
title: Install DirectAuth Swift package
---

In project settings, add DirectAuth Swift SDK's Github URL as a package
dependency.

```swift
import PackageDescription

let package = Package(
    name: "TorusSwiftDirectSDK",
    dependencies: [
        .package(name: "TorusSwiftDirectSDK", url: "https://github.com/torusresearch/torus-direct-swift-sdk", .upToNextMajor(from: "0.0.1"))
    ]
)
```
