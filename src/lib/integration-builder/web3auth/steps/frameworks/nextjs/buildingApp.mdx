title: Running your NextJS App Locally
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

To build this project locally, create a new nextjs project using the following command:

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npx create-next-app --template typescript
```

</TabItem>
<TabItem value="yarn">

```shell
yarn create next-app --typescript
```

</TabItem>

</Tabs>

Replace the file `package.json` in the root folder.

Replace the contents of the `pages` folder with the following files mentioned in the Quick Start.

Finally, replace the file `globals.css` in the `styles` folder.
