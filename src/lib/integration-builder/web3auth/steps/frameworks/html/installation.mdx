Install Web3Auth using either [npm](https://www.npmjs.com/package/@web3auth/web3auth) or CDN:

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "jsdelivr", value: "jsdelivr" },
  ]}
>
<TabItem value="npm">

```shell
npm i --save @web3auth/web3auth
```

</TabItem>

<TabItem value="jsdelivr">

```html
<script src="https://cdn.jsdelivr.net/npm/@web3auth/web3auth@0.6.3/dist/web3auth.umd.min.js"></script>
```

</TabItem>

</Tabs>
