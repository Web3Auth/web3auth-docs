---
title: Log user out
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

To log the user out from your application, call `torus.logout` or
`torus.cleanUp` to also clean up the Torus button and SDK installation.

<Tabs
  defaultValue="logout"
  values={[
    { label: "Logout", value: "logout" },
    { label: "Clean up", value: "cleanup" },
  ]}
>
<TabItem value="logout">

```js
await torus.logout();
```

</TabItem>

<TabItem value="cleanup">

```js
await torus.cleanUp();
```

</TabItem>

</Tabs>
