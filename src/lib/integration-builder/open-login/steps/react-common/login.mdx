---
title: Login with openlogin sdk
---

Call `openlogin.login` to trigger user login flow where it's appropriate in your
application lifecycle.

```js
    await sdkInstance.login({
      loginProvider: "google",
      redirectUrl: `${window.origin}/solana`,
    });
```

In this example,`openlogin.login` will redirect you to google SSO page for authentication, as here we are
using `google` as `loginProvider`

`loginProvider` is optional , if skipped then sdk will open a modal of all supported login providers like
`google`, `twitter`, `passswordless` etc.

 You can checkout [api reference](https://docs.beta.tor.us/open-login/api-reference) to know about all supported and custom login provider values.
