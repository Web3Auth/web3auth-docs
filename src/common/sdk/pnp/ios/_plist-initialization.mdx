In your application bundle add a plist file named `Web3Auth.plist` with the following information:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>ClientId</key>
  <string>YOUR_OPENLOGIN_CLIENT_ID</string>
  <key>Network</key>
  <string>sapphire_devnet</string>
  <key>RedirectUrl</key>
  <string>YOUR_REDIRECT_URL(bundleId://auth)</string>
</dict>
</plist>
```

When you have a `Web3Auth.plist` and you have specified your Web3Auth configuration there, you can
simply construct your Web3Auth instance with

```swift
import Web3Auth

let web3auth = Web3Auth()
```
