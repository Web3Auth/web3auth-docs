#### `TorusStorageLayerArgs`

This `TorusStorageLayer` constructor takes an object with `TorusStorageLayerArgs` as input. It
contains the following parameters:

<Tabs
  defaultValue="table"
  values={[
    { label: "Table", value: "table" },

  ]}
>

<TabItem value="table">

| Parameter           | Description                                                                                                            |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `enableLogging?`    | Enables logging on the console.                                                                                        |
| `hostUrl?`          | URL for connecting to your metadata server. Use `https://metadata.tor.us` for connecting to the Torus Metadata Server. |
| `serverTimeOffset?` | Time offset for letting user connect to metadata server.                                                               |

</TabItem>

<TabItem value="type">

```tsx
declare type TorusStorageLayerArgs = {
  enableLogging?: boolean;
  // Use `https://metadata.tor.us` for connecting to the Torus Metadata Server
  hostUrl?: string;
  serverTimeOffset?: number;
};
```
