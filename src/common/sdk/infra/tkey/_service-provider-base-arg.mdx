The `@tkey/service-provider-base` is a generic package that can be used to associate tKey with a
postbox key that can be used as Share A for the authentication flow. This `ServiceProvider`
constructor takes an object with `ServiceProviderArgs` as input. It contains the following
parameters:

#### `ServiceProviderArgs`

The `ServiceProviderArgs` object is the mandatory object needed to be passed within the
`ServiceProvider` constructor. It contains the following parameters:

| Parameter        | Type      | Description                                                      | Mandatory |
| ---------------- | --------- | ---------------------------------------------------------------- | --------- |
| `enableLogging?` | `boolean` | This option is used to specify whether to enable logging.        | No        |
| `postboxKey?`    | `string`  | The postbox key that can be used as the Share A of the tKey flow | No        |
