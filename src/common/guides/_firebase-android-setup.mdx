#### Configure Firebase for Android

- Once the project is created and authentication is configured. Let's use Firebase for Android by clicking on the `Android` button on the home screen
  of your project.

  ![Firebase Project Android - Set up web 1](/content-hub/guides/firebase/android/android-firebase-set-up-1.svg)

- In the next screen, enter the **Android package name** and click on **Register app** button to register the app for Android.

  ![Firebase Project Android - Set up web 2](/content-hub/guides/firebase/android/android-firebase-set-up-2.svg)
  ![Firebase Project Android - Set up web 3](/content-hub/guides/firebase/android/android-firebase-set-up-3.svg)

- In the Next screen, Download and add config file to your Android Studio Project as shown below. Then Click on `Next` button.

  ![Firebase Project Android - Set up web 4](/content-hub/guides/firebase/android/android-firebase-set-up-4.svg)

- Next screen shows ways to add **Firebase** to your application. To make the `google-services.json` config values accessible to Firebase SDKs, you
  need the Google services Gradle plugin. We will be adding the buildscript dependency to the `<project>/build.gradle` file.

  ![Firebase Project Android - Set up web 5](/content-hub/guides/firebase/android/android-firebase-set-up-5.svg)

- Then, in your module (app-level) `build.gradle` file, add both the google-services plugin and any Firebase SDKs that you want to use in your app:

  ![Firebase Project Android - Set up web 6](/content-hub/guides/firebase/android/android-firebase-set-up-6.svg)

- After adding the plugin and the desired SDKs, sync your Android project with Gradle files.

  ![Firebase Project Android - Set up web 7](/content-hub/guides/firebase/android/android-firebase-set-up-7.svg)
