---
title: Web3Auth MPC Core Kit React Native Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth MPC Core Kit in
your React Native app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/mpc-core-kit/33).

:::tip building locally

- Clone the PnP React Native Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/tree/main/mpc-core-kit-react-native/mpc-core-kit-rn-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm run ios # or npm run android
  ```

:::
