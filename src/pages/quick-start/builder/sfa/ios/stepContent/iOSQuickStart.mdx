---
title: Web3Auth SFA Swift/iOS Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth SFA in your
Swift/iOS app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/sfa-swift-sdk/30).

:::tip building locally

- Clone the PnP Swift/iOS Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/single-factor-auth-ios/sfa-ios-quick-start sfa-ios-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd sfa-ios-quick-start
  # run project in Xcode
  ```

:::

:::note Requirements

- iOS 14+, Xcode 11.4+ / 12.x & Swift 4.x / 5.x

:::
