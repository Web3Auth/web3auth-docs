title: Create and initialize the Web3Auth SFA instance

Let's now create an instance of the `SingleFactAuth`.

We need the `network` to initialize SingleFactAuth class. `network` signifies the type of
network(`mainnet`, `cyan`, `aqua`, or `testnet`) you want to initialize web3auth with.
