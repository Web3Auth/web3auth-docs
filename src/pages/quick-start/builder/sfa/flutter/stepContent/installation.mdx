---
title: Install Web3Auth
---

For making hybrid iOS & Android applications using Flutter, we have the Web3Auth's
[single-factor-auth-flutter](https://github.com/Web3Auth/single-factor-auth-flutter) SDK to
integrate Web3Auth into your application.

Add `single_factor_auth_flutter` as a dependency to your `pubspec.yaml`

```sh
flutter pub add single_factor_auth_flutter
```

or

```yaml
dependencies:
  single_factor_auth_flutter: ^latest # or the latest version
```

:::warning Update to the Latest-SDK

Please use the **[latest version](https://github.com/Web3Auth/single-factor-auth-flutter/releases)**
of Web3Auth's Flutter SDK.

:::
