---
title: Web3Auth SFA Flutter Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth SFA in your
Flutter app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/sfa-flutter-sdk/31).

:::tip building locally

- Clone the SFA Flutter Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/single-factor-auth-flutter/sfa_flutter_quick_start sfa-flutter-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd sfa-flutter-quick-start
  # run project in Android Studio or in VSCode
  ```

:::
