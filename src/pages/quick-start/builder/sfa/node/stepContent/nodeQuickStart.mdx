---
title: Web3Auth SFA Node Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Single Factor Auth
in your Node.js app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/core-kit-sfa-node/23).

:::tip building locally

- Clone the SFA Node Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/tree/main/single-factor-auth-node/sfa-node-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm start
  ```

:::
