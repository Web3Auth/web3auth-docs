---
title: Create and initialize the Web3Auth SFA instance
---

Let's now create an instance of the `SingleFactorAuth`.

We need the `network` to initialize SingleFactorAuth class. `network` signifies the type of
network(`sapphire-mainnet` or `sapphire-devnet`) you want to initialize web3auth with.

:::info sdk reference

Read more about [SFA Android SDK](/sdk/sfa/sfa-android).

:::
