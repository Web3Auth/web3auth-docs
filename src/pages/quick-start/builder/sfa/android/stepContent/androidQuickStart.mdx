---
title: Web3Auth SFA Android Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth SFA in your
Android app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/core-kit-sfa-android/26).

:::tip building locally

- Clone the SFA Android Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/single-factor-auth-android/sfa-android-quick-start sfa-android-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd sfa-android-quick-start
  # run android project in Android Studio
  ```

:::
