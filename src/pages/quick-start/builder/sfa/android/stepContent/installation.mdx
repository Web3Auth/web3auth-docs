---
title: Installation
---

Add Web3Auth's SFA Android SDK to the dependency list. In your app-level `build.gradle` dependencies
section, add the following:

```groovy
dependencies {
    implementation 'com.github.web3auth:single-factor-auth-android:0.0.6'
}
```

:::warning Update to the Latest-SDK

Please use the **[latest version](https://github.com/Web3Auth/single-factor-auth-android/releases)**
of Web3Auth's SFA Android SDK.

:::
