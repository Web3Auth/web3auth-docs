---
title: Web3Auth SFA React Quick Start
---

This guide is designed to help you quickly integrate a basic instance of Web3Auth Single Factor Auth
JS SDK in your React app.

If you face any problem anytime, you can always find help in the
[Web3Auth Community](https://web3auth.io/community/c/help-core-kit/core-kit-sfa-web/22).

:::tip building locally

- Clone the SFA React Quick Start Application

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-core-kit-examples/single-factor-auth-web/quick-starts/sfa-react-quick-start w3a-quick-start
  ```

- Install & Run

  ```shell npm2yarn
  cd w3a-quick-start
  npm install
  npm run start
  ```

:::
