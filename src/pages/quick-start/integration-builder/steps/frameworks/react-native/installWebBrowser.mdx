After that, depending on whether you are using bare React Native or Expo, you will need to install either the `expo-web-browser` or the
`@toruslabs/react-native-web-browser` package to get an implementation of the `WebBrowser` interface.

```shell
expo install expo-web-browser
```

</TabItem>

<TabItem value="bare">

```shell
npm install --save @toruslabs/react-native-web-browser
```

</TabItem>
