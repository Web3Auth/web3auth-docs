---
title: Using Web3Auth NextJS Quick Start
---

To use this NextJS Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [file an issue](https://github.com/Web3Auth/Web3Auth/issues/new/choose) for us to help
you out as soon as possible!

**To build this project locally, create a new nextjs project using the following command:**

```shell
yarn create next-app --typescript
```

- Replace the file `package.json` in the root folder.

- In the `pages/` folder with replace `App.tsx` & add your blockchain specific `.ts` file.

- Finally, replace the file `globals.css` in the `styles` folder.

**Run the app using**

```shell
npm run-script dev
```
