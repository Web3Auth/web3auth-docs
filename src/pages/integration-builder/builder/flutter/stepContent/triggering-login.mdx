---
title: Triggering the login
---

Call the `login()` method on the `Web3AuthFlutter` instance to trigger the login flow. You should pass the `loginProvider` parameter to specify the
provider you wish to use.

:::info Email Passwordless

If you are using the `email_passwordless` login provider, you will also have to specify the email address as `login_hint` in the `extraLoginOptions`.

```dart
await Web3AuthFlutter.login(
    LoginParams(
        loginProvider: Provider.email_passwordless,
        extraLoginOptions: ExtraLoginOptions(
            login_hint: "hello@web3auth.io"
        )
    )
);
```

:::
