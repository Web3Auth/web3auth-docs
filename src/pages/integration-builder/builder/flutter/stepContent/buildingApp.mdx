title: Using Web3Auth Flutter Quick Start Guide
---

To use this Flutter Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Discussions Page](https://github.com/Web3Auth/Web3Auth/discussions) for us to
help you out as soon as possible!

**To build this project locally on Android Studio or VS Code, go through the following steps:**

Create a new Flutter project following the steps from [here](https://docs.flutter.dev/get-started/codelab).

**Build and Run the app using Android Studio**

Follow [this guide](https://developer.android.com/studio/run) to run your app via Android Studio. or follow
[this guide](https://docs.flutter.dev/get-started/test-drive?tab=vscode) to run it via VS Code.
