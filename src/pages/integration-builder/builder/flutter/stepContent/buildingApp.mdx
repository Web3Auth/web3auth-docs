---
title: Using Web3Auth Flutter Quick Start Guide
---

To use this Flutter Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Community](https://web3auth.io/community/) for us to help you out as soon as
possible!

:::tip building locally

- Create a new Flutter project following the steps [mentioned here](https://docs.flutter.dev/get-started/codelab).
- Build and Run the app using [Android Studio](https://developer.android.com/studio/run). Or you can even
  [use VS Code](https://docs.flutter.dev/get-started/test-drive?tab=vscode) to run your app.

:::
