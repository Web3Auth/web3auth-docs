---
title: Using Web3Auth React Native Quick Start Guide
---

To use this React Native Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or
simply copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Discussions Page](https://github.com/Web3Auth/Web3Auth/discussions) for us to
help you out as soon as possible!

:::tip building locally

- Create and Run a new React Native project using the steps [mentioned here](https://developer.android.com/studio/projects/create-project).

:::
