---
title: Install ethers
---

Install `ethers` using either [npm](https://www.npmjs.com/package/ethers) or [yarn](https://yarnpkg.com/package/ethers):

```bash npm2yarn
npm install --save ethers
```

:::tip

Add the file `ethersRPC.ts` to the project. This file contains the code to interact with the blockchain.

:::
