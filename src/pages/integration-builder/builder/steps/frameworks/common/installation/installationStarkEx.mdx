---
title: Install StarkEx Libraries
---

Install StarkEx Integration libraries using either [npm](https://www.npmjs.com/package/@starkware-industries/starkex-js) or
[yarn](https://yarnpkg.com/package/@starkware-industries/starkex-js):

```bash npm2yarn
npm install --save web3 bn.js elliptic @starkware-industries/starkex-js @starkware-industries/starkware-crypto-utils
```

:::tip

Add the file `starkexRPC.ts` to the project. This file contains the code to interact with the blockchain.

:::
