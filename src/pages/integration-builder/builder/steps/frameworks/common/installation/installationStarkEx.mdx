title: Install StarkEx Libraries
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install StarkEx Integration libraries using either [npm](https://www.npmjs.com/package/@starkware-industries/starkex-js) or
[yarn](https://yarnpkg.com/package/@starkware-industries/starkex-js):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save web3 bn.js elliptic @starkware-industries/starkex-js @starkware-industries/starkware-crypto-utils
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add web3 bn.js elliptic @starkware-industries/starkex-js @starkware-industries/starkware-crypto-utils
```

</TabItem>

</Tabs>

:::tip

Add the file `starkex.ts` to the project. This file contains the code to interact with the blockchain.

:::
