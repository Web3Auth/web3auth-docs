title: Install StarkNet Libraries
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install StarkNet Integration libraries using either [npm](https://www.npmjs.com/package/starknet) or [yarn](https://yarnpkg.com/package/starknet):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save web3 starknet elliptic bn.js @starkware-industries/starkware-crypto-utils
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add web3 starknet elliptic bn.js @starkware-industries/starkware-crypto-utils
```

</TabItem>

</Tabs>

:::tip

Add the files `starknet.ts` & `ArgentAccount.json` to the project. These files contain the code to interact with the blockchain.

:::
