---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install StarkNet Integration libraries using either [npm](https://www.npmjs.com/package/@web3auth/web3auth) or
[yarn](https://yarnpkg.com/package/@web3auth/web3auth):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save web3 starknet elliptic bn.js @starkware-industries/starkware-crypto-utils
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add web3 starknet elliptic bn.js @starkware-industries/starkware-crypto-utils
```

</TabItem>

</Tabs>

> Add the files `starknet.ts` & `ArgentAccount.json` to the project.
