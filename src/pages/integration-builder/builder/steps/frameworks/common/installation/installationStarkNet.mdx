---
title: Install StarkNet Libraries
---

Install StarkNet Integration libraries using either [npm](https://www.npmjs.com/package/starknet) or [yarn](https://yarnpkg.com/package/starknet):

```bash npm2yarn
npm install --save web3 starknet elliptic bn.js @starkware-industries/starkware-crypto-utils
```

:::tip

Add the files `starknetRPC.ts` & `ArgentAccount.json` to the project. These files contain the code to interact with the blockchain.

:::
