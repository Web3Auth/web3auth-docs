title: Install Tezos Libraries
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install Tezos Integration libraries using either [npm](https://www.npmjs.com/package/@taquito/taquito) or
[yarn](https://yarnpkg.com/package/@taquito/taquito):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save @taquito/signer @taquito/taquito @taquito/utils @tezos-core-tools/crypto-utils
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add @taquito/signer @taquito/taquito @taquito/utils @tezos-core-tools/crypto-utils
```

</TabItem>

</Tabs>

:::tip

Add the file `tezos.ts` to the project. This file contains the code to interact with the blockchain.

:::
