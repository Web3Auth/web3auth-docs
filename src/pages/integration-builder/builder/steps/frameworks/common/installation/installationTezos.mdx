---
title: Install Tezos Libraries
---

Install Tezos Integration libraries using either [npm](https://www.npmjs.com/package/@taquito/taquito) or
[yarn](https://yarnpkg.com/package/@taquito/taquito):

```bash npm2yarn
npm install --save @taquito/signer @taquito/taquito @taquito/utils @tezos-core-tools/crypto-utils
```

:::tip

Add the file `tezosRPC.ts` to the project. This file contains the code to interact with the blockchain.

:::
