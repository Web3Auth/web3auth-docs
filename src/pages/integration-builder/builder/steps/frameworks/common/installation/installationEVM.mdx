title: Install EVM Chain Libraries
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install EVM Chain Integration libraries using either [npm](https://www.npmjs.com/package/@web3auth/web3auth) or
[yarn](https://yarnpkg.com/package/@web3auth/web3auth):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save web3 @web3auth/ethereum-provider
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add web3 @web3auth/ethereum-provider
```

</TabItem>

</Tabs>

> Add the file `evm.ts` to the project.
