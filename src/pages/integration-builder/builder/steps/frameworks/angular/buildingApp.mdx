---
title: Using Web3Auth Angular Quick Start Guide
---

To use this Angular Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply
copy paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Discussions Page](https://github.com/Web3Auth/Web3Auth/discussions) for us to
help you out as soon as possible!

**To build this project locally, go through the following steps:**

Create a new angular project using the following command:

```shell
ng new angular-web3auth-example
```

> Further, replace the contents of the `src/app/` folder with the files of same name mentioned in the Quick Start.

**Run the app using**

```shell
npm start
```
