---
title: Using Web3Auth iOS Quick Start Guide
---

To use this iOS Quick Start Guide, you can either study the steps and the different snippets of code and add them to your application, or simply copy
paste the entire files and run them on your own.

If you face any problem anytime, you can always refer to the [Web3Auth Discussions Page](https://github.com/Web3Auth/Web3Auth/discussions) for us to
help you out as soon as possible!

:::tip building locally

- Create a new iOS project following the steps [mentioned here](https://developer.apple.com/documentation/xcode/creating-an-xcode-project-for-an-app).
- Build and Run the app using [Xcode](https://developer.apple.com/documentation/xcode/building-and-running-an-app).

:::
